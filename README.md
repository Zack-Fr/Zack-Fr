## Featured work

### GuidePortal / ZackLab

GuidePortal is an AI-guided 3D product experience layer for websites. It explores how voice/text guidance, product controls, and real-time visual interaction can help users understand products without requiring a full website rebuild.

The main technical challenge was building the bridge between website UI, backend state, AI/runtime logic, and Unreal Engine presentation so that user intent could translate into reliable product and viewport behavior.

### Mia

Mia is a MetaHuman virtual assistant prototype built in Unreal Engine. The project explores real-time digital presence, conversational interaction, voice/text response behavior, and packaged deployment as a Windows executable.

### Lumea3D

Lumea3D is a browser-based 3D collaboration prototype built with React, Vite, Three.js, and WebGL. It explores shared camera viewing, scene interaction, design variants, and collaborative 3D review workflows.

### Bose Interactive

An Unreal Engine 5.5 interactive product visualization demo focused on product variants, accessories, camera transitions, UI logic, Blueprint interactions, environment design, and real-time presentation.
